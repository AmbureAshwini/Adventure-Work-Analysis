Case Study: Adventure Works Sales Analysis (Power Query)
📌 Project Overview

This project focuses on analyzing sales data from the Adventure Works dataset using Power Query in Power BI. The goal of the analysis is to clean, transform, and analyze sales data to identify key business insights related to revenue performance, product demand, and regional sales trends.

Power Query was used extensively for data cleaning, transformation, and data modeling, enabling efficient preparation of raw datasets for further analysis and visualization.

🗂 Dataset Description

The Adventure Works dataset simulates the operations of a global manufacturing company and includes multiple related tables such as:

Sales Data – Sales transactions and order details

Customer Data – Customer demographics and locations

Product Data – Product categories and subcategories

Date Table – Time-based attributes for sales analysis

Territory Data – Regional sales information

These datasets were combined using Power Query transformations to create a structured data model for analysis.

🔄 Data Preparation Using Power Query

Power Query was used to perform several data transformation and cleaning tasks, including:

Removing duplicate and null values

Renaming and formatting columns

Splitting and merging columns

Creating calculated columns

Filtering unnecessary data

Merging multiple datasets

Creating relationships between tables

These steps helped transform raw sales data into a clean and analysis-ready dataset.

Metrics Analyzed

High-performing markets
